# lab2 github

## to delete dev branch remotly :

```
git push origin --delete dev
```

## to delete dev branch locally :

```
git branch -d dev
git branch -D dev
```

## to list all tags :

```
git tag 
```

## to delete tag remotly :

```
git push origin --delete v1.7
```

## To delete tag locally :

```
git tag -d v1.7
```
